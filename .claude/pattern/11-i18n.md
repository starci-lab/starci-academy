# 11 — i18n (next-intl)

## Config — `src/i18n/`
- `routing.ts` — khai báo locales + defaultLocale (locale routing prefix `[locale]`).
- `navigation.ts` — wrappers `Link`, `redirect`, `useRouter`, `usePathname` aware locale. **Dùng các wrapper này, không dùng `next/link` raw.**
- `request.ts` — load messages theo locale cho server.

## Strings — `src/messages/`
- `en.json`, `vi.json` — key dịch. Cùng cấu trúc 2 file.
- Dùng trong component: `useTranslations("namespace")` (next-intl) → `t("key")`.
- Cũng có `i18next/react-i18next` cho 1 số chỗ.

## Quy tắc
- Thêm text hiển thị → thêm key cả `en.json` + `vi.json` (đừng hardcode chữ trong JSX).
- Namespace theo feature (vd `aiSettings`, `course`, `auth`…).
- Route mọi nơi giữ locale prefix → build qua `pathConfig()` (xem 10) + Link từ `i18n/navigation`.
