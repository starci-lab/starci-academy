# drafts/ — rule mới / feedback chờ merge

Mỗi lần thầy dạy/chốt/sửa 1 điều, hoặc feedback sau khi trò làm xong → **1 file `<temp-name>.md`** ở đây
(KHÔNG sửa `main.md`/`starci-<element>.md` trực tiếp). Mỗi file 1 ý, gọn:

```
# <tên ý>  (ngày)
- File/§ đích: main.md §X  HOẶC  starci-<element>.md
- Bài học: <thầy dạy/feedback gì>
- Luật mới: <phát biểu STRICT> + nguyên nhân gốc + cách đúng
```

- Drafts coi như `alwaysApply`, **mới nhất thắng** khi mâu thuẫn main/element.
- `/fe` đọc & áp main + starci-* + tất cả drafts.
- `/merge` (chỉ khi thầy gõ) → fold drafts vào đúng file (mindset/luật chung → `main.md`; styling 1 element →
  `starci-<element>.md`) → xoá drafts đã gộp.
