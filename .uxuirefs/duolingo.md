# Duolingo — gamification gold standard

> Chuẩn mực game-hóa học tập. Pull 2026-06-18. `main.md` §1 đã dặn "dashboard→Duolingo". Tham khảo **gamification + hub**.

## Pattern lấy được
- **Streak** — đếm ngày liên tiếp, tạo thói quen (StarCi có streak strip).
- **XP** — điểm cho mỗi lesson/achievement (StarCi có XP/points).
- **League/Leaderboard** — xếp hạng tuần đấu với peer (StarCi có league).
- **Learning path map** — cây tiến độ trực quan: đã xong / đang ở đâu / sắp tới.
- **Daily goal** — mục tiêu XP/ngày để giữ streak (StarCi có daily-quest).
- **Hearts/Lives** — giới hạn lượt sai → tạo "stakes" + lý do quay lại.
- **Achievements** — badge mốc (streak/lesson count/speed).
- **Home/Dashboard** — hub: streak + rank league + daily goal + next lesson.
- **Nudges** — nhắc mất streak / đua league / hoàn thành daily.
- **Mascot + microcopy thân thiện** — Duo cú; ngôn ngữ động viên, nhân-hoá.

## → Áp cho StarCi
- StarCi đã có **streak/XP/league/daily-quest** → Duolingo là chuẩn để **đánh bóng** (ring streak, path map trực quan, nudge).
- **Learning path map trực quan** = cơ hội: StarCi có MindMap + content-map nhưng chưa có "path map kiểu Duolingo" (chấm tròn tuần tự, đã-xong/đang/khoá). Có thể làm cho trang Chỉ mục/khóa.
- **Mascot + microcopy thân thiện** → StarCi đang làm **mascot ngọn lửa cho FAB AI** + có thể dùng mascot xuyên suốt (dashboard/empty-state) như Duo.
- **Daily nudge + hearts** = cơ chế giữ chân (StarCi có daily-quest; "hearts/lives" có thể map sang "lượt AI/challenge ngày" — khớp model quota đang bàn).
- **Dashboard = hub** (streak+rank+daily+continue) → StarCi dashboard đã rework theo hướng này.
