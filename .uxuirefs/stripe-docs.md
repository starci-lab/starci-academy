# Stripe docs — docs gold standard (3-pane + code + microcopy)

> Docs chuẩn mực ngành. Pull 2026-06-18. Tham khảo khu **Đọc bài + nav** StarCi.

## Pattern lấy được
**Nav TRÁI:** phân cấp section (Payment options → "Most popular"/Online/In-person); **active highlight**; **collapse
section header** (gom tải nhận thức); progressive disclosure (category mở ra link cụ thể).
**Content:** breadcrumb; H2/H3 anchor; **callout "Most popular"** đẩy task ưu tiên lên đầu; link gom theo semantic group.
**Typography:** reading width ~600–800px; H1/H2/H4 rõ; **nhiều whitespace** giữa section (dễ quét).
**On-this-page PHẢI:** TOC anchor H2/H3, sticky.
**Search/AI:** "Find your use case" CTA; **"Build on Stripe with AI"** ở header (AI entry).
**Microcopy:** **động từ hành động** ("Get started/Set up/Integrate"); badge "No code required"; link giải thích **TẠI SAO**
("Faster checkout with Link"); "Quickstart" = đường nhanh nhất.

## → Áp cho StarCi
- **Collapse section + whitespace nhiều** giữa khối → StarCi content-map + body học cách giảm tải.
- **Callout "Most popular"/đề xuất** → StarCi có thể đánh dấu bài/khóa nên-học-trước.
- **AI entry ở header** ("Build with AI") → StarCi để FAB AI (đã làm) + cân nhắc 1 entry ở search/header.
- **Microcopy động-từ + "tại sao"**: nâng copy nút/link StarCi (hiện nhiều label generic).
- Reading width 600–800px (như react.dev) — củng cố ý hẹp cột đọc.
